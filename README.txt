# Random Graphs Simulation (Erdős–Rényi)

This project simulates random graphs using the G(n, p) model and analyzes:

- Graph connectivity
- Emergence of the giant component

## Key Results

We observe two phase transitions:

- Giant component: p ≈ 1/n
- Connectivity: p ≈ ln(n)/n
